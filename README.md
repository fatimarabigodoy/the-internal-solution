# The Internal Solution

An interactive choose-your-own-adventure experience showcasing how Algonquin College advertising students solve agency challenges.

## About

This game simulates agency chaos scenarios where employers can experience firsthand how advertising students from Algonquin College can handle:
- Last-minute rebrands
- High-stakes social media campaigns  
- Complex pitch presentations

Players make hiring decisions and see how different student roles (Copywriter, Designer, Strategist, Digital Strategist, Account Coordinator, Researcher) collaborate to deliver winning work.

## How to Play

[View the live game here]([#](https://the-internal-solution.vercel.app/)) *(link will be added after deployment)*

Or run locally:
1. Download this repository
2. Open `index.html` in your web browser
3. Click "BEGIN" and follow the prompts
